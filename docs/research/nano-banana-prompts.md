# Nano Banana Prompt Pack

Use these prompts for Pink Hunter visual assets.

## Global Art Direction
- Style: hand-drawn botanical illustration
- Output: PNG with transparent background
- Visual tone: soft spring, refined, premium, mobile-product friendly
- Line work: clean ink outline with watercolor fill
- Avoid: photorealistic rendering, 3D, mockup framing, background scene, text unless explicitly requested

## Global Positive Prompt
`hand-drawn botanical illustration, pastel spring palette, pink blossom theme, soft watercolor wash, clean ink outline, elegant negative space, high-end product illustration, transparent background, PNG, no watermark`

## Global Negative Prompt
`photorealistic, 3d render, cluttered background, mockup, typography, watermark, logo sheet, busy composition, muddy colors, dark horror style, low contrast, pixelated`

## 1. Brand Logo Mark
### Prompt
`Create a hand-drawn logo mark for "Pink Hunter", a mobile-first blossom tree map focused on cherry, plum, peach, magnolia, and crabapple trees. The mark should feel elegant, warm, spring-like, and easy to recognize at small size. Use a soft pastel pink palette with subtle contrast, refined ink outlines, and a premium editorial-botanical feel. Build the symbol from blossom petals and gentle map-discovery cues, without looking like a generic location pin. Transparent background, PNG.`

## Recommended Export Notes
- Generate each asset individually.
- Ask for transparent PNG output.
- Keep backgrounds empty.
- For map usage, request strong silhouette and minimal tiny details.
- For comparison illustrations, request generous spacing so the assets can be cropped into cards later.
